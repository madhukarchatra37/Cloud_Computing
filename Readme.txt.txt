DocWordCount.java:-
Counts the frequency of each word in the files
The Input folder and Output folder are the arguments to this program

TermFrequency.java:-
Counts Logarithamic Term frequency of each word
The Input folder and Output folder are the arguments to this program

TFIDF.java:-
It is associated with two mapper and two reducer class
WFM.java and WFR.java are the respective mapper and reducer class for word count
IDFM.java and IDFR.java are the respective mapper and reducer class for counting term frequency
The Input folder and two Output folders are the arguments to this program
